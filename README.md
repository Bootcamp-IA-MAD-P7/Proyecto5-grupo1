# Fall-Sentinel (Proyecto5 — Grupo 1)

Sistema de detección de caídas mediante Machine Learning: app móvil Flutter + API FastAPI.

## Estructura del repositorio

```
├── Frontend/     # App Flutter (Fall Detector Tester)
├── Backend/      # FastAPI, ML, notebooks Kaggle y datasets
├── docs/         # Documentación del proyecto
│   └── daily/    # Daily standups del equipo
├── .specify/     # Orquestación IA y especificaciones SDD
└── .github/      # CI/CD (Firebase App Distribution)
```

## Inicio rápido

### Frontend (Flutter)

```bash
cd Frontend
flutter pub get
flutter run
```

### Backend (pendiente de implementación)

```bash
cd Backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

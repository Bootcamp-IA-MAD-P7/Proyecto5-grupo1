**Acción en el móvil (1 min):**

2. Activar **Instalar vía USB** (o desactivar optimización MIUI)
3. Instalar `Descargas/sentilife-v1.apk` (tocar el archivo) **o** aceptar el popup de `adb install`

1. Abrir SentiLife → `checkForUpdate()` detecta `1 < 100`
2. Diálogo **Actualización disponible** (i18n T3.7)
3. Pulsar **Actualizar ahora** → descarga desde LAN `:8765`
4. Instalador Android → confirmar → app queda en v100

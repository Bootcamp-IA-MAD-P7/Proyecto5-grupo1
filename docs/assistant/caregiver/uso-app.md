# Guía para cuidador

## Personas monitorizadas
Desde el inicio puedes ver y gestionar las personas que cuidas. Añade una persona con el flujo de vinculación (código/enlace) y asegúrate de que acepte el consentimiento.

## Alertas
Cuando el sistema detecta una posible caída, recibes una notificación. Ábrela, revisa el detalle y marca si fue una caída real o una falsa alarma. Ese feedback mejora el sistema.

## Historial
Consulta alertas recientes y su estado (pendiente, confirmada, descartada). Filtra por persona si tienes varias vinculadas.

## Notificaciones push
Activa las notificaciones del sistema para SentiLife. Sin ellas puedes perder avisos urgentes. Si cambias de teléfono, vuelve a iniciar sesión para registrar el dispositivo.

## Privacidad
Solo ves datos de las personas vinculadas a ti. No compartas códigos de emparejamiento fuera de un canal seguro.

# DS-01 — SisFall (crudo)

Colocar aquí los archivos `.txt` originales de SisFall (uno por ensayo).
Patrón de nombre: `<CODE>_<SUBJECT>_R<TRIAL>.txt` (ej. `F05_SA01_R04.txt`).

Fuente: http://sisfall.imed.li/

Procesar a CSV con:
`python ml/build_sisfall_dataset.py --root data/raw/sisfall --out data/processed/sisfall_dataset.csv`
